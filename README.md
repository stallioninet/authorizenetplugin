# authorizenetplugin
WordPress plugin for Authorize.net payment gateway. 

The plugin is a simple one which can be used in any WordPress site for payments through Authorize.Net
This is the only available WordPress plugin for Authorize.net

After installation, just call the payment function and pass the necessary arguments.
Once payment is done, the function will return a success message and payment transaction details, which can be 
captured and stored/displayed as required.
